Available Commands:

  • add room <number> - Add a new room
  • delete room <number> - Delete a room
  • add reservation <room> <arrival> <departure> <guest name> - Add reservation (date format: MM/DD/YYYY)
  • delete reservation <room> <arrival> <departure> <guest name> - Delete reservation
  • print reservations all - Show all reservations
  • print reservations room <number> - Show reservations for a room
  • print reservations guest <name> - Show reservations for a guest
  • print reservations arrival <date> - Show reservations by arrival date
  • print reservations departure <date> - Show reservations by departure date
  • clear - Clear the terminal
  • help - Show this help message
